# ALX Higher-Level Programming

Python programming exercises completed as part of the ALX software engineering curriculum. This repository focuses on higher-level programming, scripting and problem solving.

## What This Covers

- Python syntax and control flow
- Data structures and functions
- Object-oriented programming
- File handling and exceptions
- Test-driven development basics

## How to Use

Browse the topic folders and run the Python scripts from the command line. Some exercises may include test files or expected output examples.

```bash
python3 script.py
```

## Tools

Python, shell, Git and basic testing utilities.
